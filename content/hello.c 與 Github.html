了解如何執行最簡單的 .c 程式, 了解如何修改 .c 程式碼.

了解如何利用 git 指令, 在新增、提交、推送不同 .c 版本之後, 如何擷取不同版本 .c 程式碼的資料.

程式開發過程, 如何有效管理各階段資料的歷程. 尤其當多人協同進行程式開發時, 如何精確呈現各改版內容間的差異, 讓各成員藉以迅速因應配合.
